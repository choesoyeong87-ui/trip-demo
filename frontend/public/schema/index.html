Membership

회원 / 등급 / 호스트 구조

MEMBER_GRADE

등급 기준

회원 등급과 혜택 기준을 관리하는 마스터 테이블

PKGRADE_NO
핵심 컬럼GRADE_CODE, GRADE_NAME, MIN_TOTAL_AMOUNT, MILEAGE_RATE, STATUS
용도등급 계산, 혜택 기준, 정렬 순서 관리

MEMBER

회원

기본 회원 정보와 보유 마일리지, 활성 여부 관리

PKUSER_NO
FKGRADE_NO → MEMBER_GRADE
핵심 컬럼USER_ID, USER_NAME, PHONE, MILEAGE, EMAIL, ENABLED

HOST_PROFILE

호스트

사업자 정보와 승인 상태를 별도 관리하는 호스트 프로필

PKHOST_NO
FKUSER_NO → MEMBER, APPROVED_BY → MEMBER
핵심 컬럼BUSINESS_NAME, BUSINESS_NUMBER, APPROVAL_STATUS, APPROVED_AT

Lodging

ACCOMMODATION

숙소 공통 정보

PKACCOMMODATION_NO
FKHOST_NO → HOST_PROFILE
핵심 컬럼ACCOMMODATION_NAME, ACCOMMODATION_TYPE, ADDRESS, LATITUDE, LONGITUDE, STATUS
포인트예약 대상 자체가 아니라 “숙소 공통 정보” 계층

Caution
